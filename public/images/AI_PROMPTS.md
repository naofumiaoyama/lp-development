## Image Generation Workflow

### Recommended AI Tools:

1. **DALL-E 3** (via ChatGPT Plus)
   - Best for: Detailed prompts, high-quality business illustrations
   - Output: PNG format, good resolution control
   - Advantage: Follows complex instructions well

2. **Midjourney v6**
   - Best for: Artistic style, consistent aesthetics
   - Output: High resolution, professional quality
   - Advantage: Style consistency across multiple images

3. **Stable Diffusion (via Automatic1111 or ComfyUI)**
   - Best for: Fine control, local generation, no API costs
   - Output: Customizable resolution and format
   - Advantage: Complete control over generation parameters

### Generation Steps:

1. **Select the image to generate** from this prompt list
2. **Choose an AI tool** based on your access and requirements
3. **Copy the appropriate prompt** (standard or alternative)
4. **Generate the image** using the AI tool
5. **Download and save** to `c:\PrimaMateriaWeb\lp-development\public\images\`
6. **Optimize file size** if needed (use TinyPNG or ImageOptim)
7. **Update component imports** in the codebase

### Post-Processing:

- **PNG files:** Optimize with TinyPNG or ImageOptim
- **SVG files:** Clean up with SVGOMG or Figma export
- **Background removal:** Use Remove.bg or Photoshop if needed
- **Color adjustment:** Ensure exact color match (#1a1d23, #14b8a6)

### File Naming Convention:

- Use lowercase with hyphens: `hero-background.png`
- Use descriptive names: `usecase-rag.png` not `image1.png`
- Keep consistent naming across all assets
